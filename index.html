Happy Birthday Animation

This birthday greeting is animated using various methods with fallbacks for older browsers. The SMIL animations are called for SVG objects using Velocity.js. There are PNG image fallbacks for browsers that do not support SVG images. Fallbacks for animation are provided through CSS transforms and animations as well as CSS sprite sheet animation. Support is detected using Modernizr, and the appropriate solution is served via JavaScript
