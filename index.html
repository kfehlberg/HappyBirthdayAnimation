Happy Birthday Animation

This birthday greeting is animated using various methods with fallbacks for older browsers. The svg animations are called using Velocity.js. There are png fallbacks for browsers that do not support svg. Fallbacks for animation are provided through CSS transforms and animations as well as CSS sprite sheet animation. Support is detected using Modernizr, and the appropriate solution is served via JavaScript
